# ProTeX

A multi-platform LaTeX editor with embedded Python, command shortcuts, a bespoke HTML and Markdown to LaTeX converter, and mandatory accessibility.

 - Embedded Python
   - Documents can contain escapes wrapping Python code that gets run at compile time. It's output is included in the final PDF.
   - Each section can reference the ones that came before them.
 - Command Shortcuts
   - Type in a short sequence mapped to a given command and have it automatically replaced by that command.
 - HTML and Markdown Converter
   - Well-formatted HTML and Markdown code can be translated to equivalent LaTeX code via the Advanced Paste feature.
 - Mandatory Accessibility
   - All built-in templates are PDF/UA compliant.
   - Any images inserted via the Edit menu require alt text.
